# Phase 3: Classical Machine Learning

**Days:** 036 – 060

---

## Overview

Learn and apply classical ML algorithms using Scikit-Learn and XGBoost, with direct applications to geotechnical prediction problems.

## Topics Covered

- Linear, Polynomial, Ridge & Lasso Regression
- Logistic Regression, KNN, Decision Trees
- Random Forest, Gradient Boosting, XGBoost
- SVM, Naive Bayes, Clustering (K-Means, DBSCAN)
- Cross-Validation, Hyperparameter Tuning, Optuna
- Model Evaluation, Ensemble Methods, SHAP
- Geotech ML: Liquefaction, Vs, Bearing Capacity, Soil Classification
- Mini Project: Full Geotech ML Pipeline
