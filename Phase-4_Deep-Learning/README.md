# Phase 4: Deep Learning

**Days:** 061 – 085

---

## Overview

Dive into neural networks with PyTorch — from basic NNs to CNNs, RNNs, LSTMs, PINNs, and Transformers — applied to geotechnical and seismic data.

## Topics Covered

- Neural Networks from Scratch & PyTorch Basics
- Activation Functions, Loss Functions, Optimizers
- CNNs (Fundamentals, Image Classification, Transfer Learning)
- RNNs, LSTM, GRU, Time Series Forecasting
- Autoencoders, VAEs, GANs
- Physics-Informed Neural Networks (PINNs)
- Graph Neural Networks, Transformers, TabNet
- Geotech DL: Seismic Signal Classification, Ground Motion Prediction
