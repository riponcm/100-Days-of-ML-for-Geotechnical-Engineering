# Phase 2: Data Science Essentials

**Days:** 016 – 035

---

## Overview

Master the core data science stack — NumPy, Pandas, Matplotlib, Seaborn, Plotly, and SciPy — with emphasis on engineering data workflows.

## Topics Covered

- NumPy (Arrays, Linear Algebra, Signal Processing)
- Pandas (DataFrames, Cleaning, GroupBy, Merging)
- Matplotlib & Seaborn (Static Visualization)
- Plotly (Interactive Visualization)
- SciPy (Optimization, Curve Fitting, FFT)
- EDA, Feature Engineering, Preprocessing
- Dimensionality Reduction (PCA, SVD)
- Mini Project: SPT Borehole Data EDA
